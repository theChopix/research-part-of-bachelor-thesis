Javascript Based Visualizations

As part of the research, we also focused on visualization methods of decision trees that are not based on python. one technique created with the help of a javascript library was presented.


D3.js

D3.js is a JavaScript library for manipulating and visualizing data using web standards such as HTML, CSS, and SVG. It provides a powerful set of tools for creating interactive and dynamic data-driven visualizations in the browser. Using d3.js and modifying the pattern available on the web, the model could be visualized with interactive elements.


The visualization at its beginning has only the first three nodes expanded. In the individual nodes, we can see the number of instances that pass through them further divided by percentage according to the target feature together with the index of split feature and threshold.


After clicking on individual nodes, it is possible to expand the tree structure and get to the leaves and its values.


Implementation of this particular method is not included in the project.
